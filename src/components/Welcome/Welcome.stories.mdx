# 这是一套使用 React+typescript 制作的组件库
用法和参数大量参考了 vikingship 和 AntD。
我为了提高自己对React的熟悉程度，深层次理解各类hook用法，才创建了这个项目，
随着React版本的更新，这套组件将成为我试验新版本特性的工具
### 安装试试
~~~javascript
npm install roundface_shan --save
~~~
### 使用
~~~javascript
// 加载样式
import 'roundface_shan/dist/index.css'
// 引入组件
import { Button } from 'roundface_shan'
~~~
